# Todo List App

A simple and interactive Todo List application built using TypeScript, HTML, and CSS.  
The app helps users manage daily tasks by allowing them to add, complete, and delete todos.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive user interface

---

## Tech Stack

- TypeScript
- HTML
- CSS
